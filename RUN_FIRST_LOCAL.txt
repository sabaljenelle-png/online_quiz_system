ONLINE QUIZ SYSTEM - RUN FIRST LOCAL

1. Open PowerShell inside this project folder.
2. Run these commands:

composer install
npm install
php artisan key:generate
php artisan migrate:fresh --seed
npm run build
php artisan serve

3. Open:
http://127.0.0.1:8000
